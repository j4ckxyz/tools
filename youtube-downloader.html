How this works. yt-dlp can't run inside a browser, so this tool sends your link to an open-source cobalt processing instance, which returns the finished MP3/MP4. Most public instances now require authentication and go up and down often — for reliable use, either add an instance + API key below, or self-host cobalt in one command (docker run ghcr.io/imputnet/cobalt) and point this tool at it. Browse public instances at instances.cobalt.best. Your settings are saved locally in this browser only.
